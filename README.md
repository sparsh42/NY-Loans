# NY-Loans
An exploratory data analysis and classification prediction of Kaggle Dataset [NY Corporate Loans](https://www.kaggle.com/theforcecoder/new-york-city-corporate-loans)
# Final Results (Accuracy)
- **RFC** - 87.31%
- **DTC** - 90.75%
- **XGB** - 92.48%
